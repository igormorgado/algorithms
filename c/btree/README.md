## Binary tree

This code implement binary tree and queue data structures.


btree.h: Binary Tree
node.h: Node data structure for Btree and Queue
queue.h: Queue using sequential arrays
queuell.h: Queue using linked lists.

To compile just type:

```
# make
# ./bt
```

To test run:

```
make valg
make clang
```
